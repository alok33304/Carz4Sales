# Carz4Sales
The *Car Sales Website* is a complete web application that makes it easy to buy and sell cars. It offers a simple and easy-to-use platform where users can browse different car listings, compare models, and choose the right car.
Technologies Used
[HTML, CSS, JavaScript].
